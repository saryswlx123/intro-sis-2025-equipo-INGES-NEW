# Guía rápida de Markdown

Esta guía muestra ejemplos básicos que puedes usar en la documentación del proyecto.

"## 1. Títulos
"# Título H1
"## Título H2
"### Título H3
"#### Título H4

## 2. Énfasis
**negrita**, *cursiva*, ~~tachado~~, `código en línea`.

## 3. Listas
- Elemento A
- Elemento B
  - Sub-elemento

1. Paso uno
2. Paso dos
3. Paso tres

## 4. Enlaces
Enlace en línea: [GitHub](https://github.com)
